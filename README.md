# 📈 Crypto Tracker App

A modern and responsive cryptocurrency tracking application built using **React**. This app allows users to view real-time crypto prices, market trends, and detailed coin information.

---

## 🚀 Features

* 💰 Live cryptocurrency prices
* 📊 Market data (market cap, volume, price change)
* 🔍 Search cryptocurrencies by name
* 📈 Individual coin details page
* ⚡ Fast and responsive UI
* 🌐 Real-time API integration

---

## 🛠️ Tech Stack

* React (Vite / CRA)
* JavaScript (ES6+)
* CSS / Tailwind (if used)
* Crypto API (e.g., CoinGecko / CoinMarketCap)
